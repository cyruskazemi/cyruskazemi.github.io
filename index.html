Hello!

My name is Cyrus Kazemi and I am a front-end web developer.

Core competencies: HTML5, CSS3, JavaScript, jQuery

Experience with: Ruby on Rails, PHP, WordPress, C#, Java

Currently learning: Ember.js, SASS

Tools I use: TextMate, Terminal, Chrome DevTools, Git, Visual Studio, CS6

I have been fascinated with computers as creative interfaces since I was young. I started making short films with friends in middle school, mimicking green screen and slow motion effects found in the likes of The Matrix.

On a whim, I decided to learn HTML and CSS and make a website for my senior project. I instantly fell in love with web development and haven't looked back. Since then, I have built websites and web applications in PHP, Java, Ruby on Rails and C#. Web development marries the creative expression that drives me with methodical problem solving provided by programming. Left brain and right brain working in harmony to produce simple, elegant and intuitive solutions.

Nowadays, I am focusing my efforts on the interactive and front-end parts of the web. I am in my element writing clean JavaScript and obsessing over minute details of a responsive design.
